안녕하세요! 승승장군 퀀트 엔진 사용자 여러분.
V27 업데이트부터는 제가 깃허브에 최신 코드를 올리면 👉 여러분은 텔레그램에서 **/update** 버튼 하나만 누르시면 됩니다. 버튼 클릭 한 번에 여러분의 서버가 스스로 제 깃허브에서 코드를 다운로드받고, 스스로 뇌(메모리)를 재부팅하여 최신 버전으로 락온(Lock-on)하는 **'자가 업데이트 엔진'**이 완성되었습니다.

각자의 서버 환경이 다르더라도 이 엔진이 완벽하게 작동할 수 있도록, **[서비스 이름 환경변수 설정]** 단계를 포함한 최종 세팅 가이드를 전달드립니다. 딱 한 번만 설정해두시면 앞으로의 패치는 1초 만에 끝납니다!

### 🛠️ [ 단 1회 필수! GCP 서버 ↔ 깃허브 통로 개설하기 ]
구글 클라우드(GCP)의 SSH 터미널을 열고 아래 명령어들을 순서대로 복사해서 엔터를 쳐주세요.

**0. [사전 준비] 서버에 Git 엔진 설치하기**
*(※ 서버에 깃(Git)이 안 깔려 있을 수 있으므로, 아래 명령어를 먼저 입력하여 설치해 줍니다.)*

sudo apt-get update
sudo apt-get install git -y

**1. [핵심] 본인의 봇 서비스 이름을 환경변수에 저장하기**
*(※ 업데이트 후 자동 재시작을 위해 꼭 필요합니다. 아래 mybot 자리에 여러분이 서버에 설정해둔 실제 서비스 명칭을 넣으세요.)*

# 1) 환경변수 설정 (따옴표 안에 본인의 서비스 이름을 넣으세요. 예: mybot)
echo 'export daemon_name="mybot"' >> ~/.bashrc

# 2) 설정 즉시 반영
source ~/.bashrc

# 3) 확인 (본인 봇 이름이 정상 출력되는지 확인)
echo $daemon_name

**2. 봇이 설치된 폴더로 이동합니다.** *(※ 본인의 봇 설치 경로가 다르다면 본인 경로에 맞게 수정하세요)*

cd /home/pipios4006

**3. Git을 초기화하고 기존 연결을 지워줍니다.** *(※ 에러가 나더라도 무시하고 진행하세요)*

git init
git remote remove origin

**4. 승승장군 공식 깃허브 주소를 연결합니다.**

git remote add origin https://github.com/pipios4006-boop/KIS-API-Python-Trading-Bot-Example.git

**5. 최신 코드를 강제 다운로드(덮어쓰기) 합니다.**

git fetch origin
git reset --hard origin/main

*(※ 화면에 HEAD is now at... 문구가 뜨면 성공입니다!)*

**6. 마지막으로 수동 재시작하여 환경변수를 인식시킵니다.**

sudo systemctl restart (자신 봇 이름)

### 🛠️ [ 참고: 엔진 코드(plugin_updater.py)의 변화 ]
이번 업데이트를 통해 봇 내부의 업데이트 엔진은 여러분이 설정한 환경변수를 자동으로 읽어오도록 설계되었습니다.

# plugin_updater.py 핵심 로직 일부

def restart_daemon(self):
    import os
    # 환경변수 'daemon_name'을 읽어오고, 만약 셋팅이 안 되어있다면 기본값 'mybot' 사용
    service_name = os.getenv("daemon_name", "mybot")
    subprocess.run(["sudo", "systemctl", "restart", service_name])

### 🎯 [ 세팅 완료! 앞으로의 패치 적용 방법 ]
이제 모든 준비가 끝났습니다! 앞으로 제가 **"패치되었습니다!"**라고 공지를 올리면 여러분은 더 이상 터미널이나 파일질라를 켤 필요가 없습니다.
 1. 텔레그램 봇 채팅방에서 **/update** 명령어를 누릅니다.
 2. 봇이 스스로 깃허브 코드를 당겨오고(git reset --hard), 환경변수에 저장된 본인의 서비스 이름을 찾아 스스로 재부팅합니다.
 3. **"✅ [동기화 완료]"** 메시지가 뜨면 업데이트 성공!
가장 진보된 자율주행 업데이트 시스템과 함께 더욱 편리한 퀀트 투자를 즐기시기 바랍니다. 감사합니다! 🚀
